# Wessage 
 Web chat application using Rails with ActionCable and React with Material UI

Requirements:
rails '6.0.0'
ruby '2.6.5'

## Installation
Install the required gems:
```bash
bundle install
```
Install dependencies:
```bash
yarn install
```
Run the migrations:
```bash
rails db:migrate
```
Run the app in localhost:3000:
```bash
rails server or rails s
```
